Predictive Analytics for Customer Churn Using R

Description:
• Developed a predictive analytics model in R to identify customer churn in subscription-based services, leveraging machine learning techniques such as logistic regression and random forests.
• Performed data preprocessing and feature engineering using dplyr and tidyr, improving data quality and enhancing model accuracy.
• Conducted exploratory data analysis (EDA) with ggplot2 and plotly to visualize customer behavior trends and key churn factors.
• Deployed an interactive Shiny dashboard for real-time churn prediction and visualization, enabling data-driven decision-making for retention strategies.

Tags: R, Machine Learning, dplyr, tifyr, EDA, Data visiualization
