Daily Affirmation API – RESTful Backend

Description:
Affirmation App is a personal full-stack project developed to explore end-to-end web application development.
The frontend is built with HTML, CSS, and JavaScript to provide an interactive interface for generating daily affirmations.
The backend is implemented using Node.js and Express, exposing RESTful APIs that return affirmations in JSON format and handle CORS and email functionality via NodeMailer.
The application is deployed with the frontend hosted on GitHub Pages and the backend on Node.js hosting, demonstrating complete development and deployment workflow.

Tags: JavaScript,Node.js, Express, RESTful API, HTML, CSS
